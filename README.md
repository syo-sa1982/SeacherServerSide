# SeacherServerSide
Unityでサーバーサイドと連携して何かを作る実験

Unity側のコードはこちら
https://github.com/syo-sa1982/SearcherClient

## 使ってるライブラリ
- gorm https://github.com/jinzhu/gorm
- goji https://github.com/zenazn/goji


## 作る予定の機能
- ユーザー情報のメンテナンス
- ユーザーが作った探索者の管理
- Client側にjson形式のAPIを投げる
